# Payload para S936B (Samsung Galaxy S25+)

- **Dispositivo:** Samsung Galaxy S25+
- **Codename:** S936B
- **Modelo:** SM-S936B
- **Firmware:** BP4A.251205.006.S936BXXUACZF1
- **Android Version:** 16 (One UI 8.5)

## Descripción
Este payload aporta los datos de configuración iniciales, metadatos y registros del sistema para la variante SM-S936B con firmware BP4A.251205.006.S936BXXUACZF1.

## Archivos Incluidos
- `metadata.json`: Metadatos del dispositivo y builds probados.
- `install.sh`: Script esqueleto para validación/empaquetado.
- `logs/getprop.txt`: Propiedades del sistema extraídas por ADB.
- `logs/uname.txt`: Identificación y versión del kernel de Linux.

## Estado del Dispositivo de Prueba
- Bootloader: Bloqueado (Locked)
- Estado Root: No rooteado (Stock Firmware)
